# Active Directory Home Lab

## Overview
This project demonstrates a Windows Server Active Directory home lab.
The lab includes domain controller setup, client machine domain join,
user and group management, Group Policy configuration, and basic
PowerShell automation.

## Lab Environment
- Windows Server 2019 (Domain Controller)
- Windows 10 Client
- Virtualization: VirtualBox / VMware
- Network: Internal / NAT Network

## Features Implemented
- Active Directory Domain Services (AD DS) installation
- Domain Controller configuration
- Client machine joined to the domain
- Organizational Units (OUs)
- User and group management
- Group Policy Objects (GPOs)
- Shared folders with NTFS permissions
- Basic PowerShell automation

## Tools & Technologies
- Windows Server
- Active Directory
- Group Policy Management
- PowerShell
- Git & GitHub

## What I Learned
- How enterprise Active Directory environments are structured
- How Group Policies affect client machines
- Importance of role-based access control
- Basics of automating administrative tasks with PowerShell
